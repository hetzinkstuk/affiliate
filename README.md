# Het Zinkstuk | Affiliate

Cross-browser plugin for Microsoft Edge, Chrome, Opera and Firefox that notifies and (automatically) redirects the user to an affiliate link when visiting one of the affiliates of [Het Zinkstuk](https://www.hetzinkstuk.nl) in one of the supported languages: Dutch, English, German, Spanish and French.
